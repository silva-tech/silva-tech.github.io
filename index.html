Product Architecture

Sensors gather user and environmental data, and upload the data through the Connection Layer.
The Decision Layer receives the information, determines the best action to take, and deploys the commands to the Device Layer.
Domestic Appliances can then render the home to a personalized optimal state.

Device Layer

Sensors gather user and environmental data; appliances execute the commands.

Connection Layer

Upload sensor data to the Decision Layer, and deploy commands to the Device Layer.

Decision Layer

Decide best possible action based on sensory information and user history.
